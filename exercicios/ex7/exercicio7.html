EX 7 - O campeão

Proposta

O campeão Construa uma aplicação que solicite nome e nota de diversas pessoas (1 até N) até que o usuário clique no botão “Quem é o campeão”. Exiba em tela o nome e a nota do campeão. OBS.: Utilize métodos da Math API para solucionar o caso.

Exercício

  1. JS: Tive problemas com o array e objeto, dificuldade muito alta para trabalhar com eles.
  2. JS: Não vi motivos ou local apropriado para a aplicação da Math API.
