Tipos de dados

Javascript possui 8 tipos de dados:

  • string
    • letras, palavras ou frases, até números, mas tudo deverá ser escrito entre áspas duplas ou simples.
  • number
    • números inteiros, quebrados (usar ponto ao invés de vírgula), exponenciais (123e-5 para 0.00123). Também existe NaN e infinidade, que será melhor abordado em um arquivo próprio.
  • bigint
    • É um formato para armazenar valores inteiros grandes demais para serem representados por um JS normal, por exemplo:
    let x = BigInt('112345567890123456789023456789')
  • boolean
    • verdadeiro ou falso
  • undefined
    • let x; (indefinido), tudo aquilo que não for definido cairá aqui. Podendo até ser atribuído o valor udefined, como em let y = undefined;
    undefined é o valor primitivo utilizado quando uma variável não teve valor algum atribuído.
  • null
    • ausência intencional de valor, por exemplo, em um objeto = {} é equivalente a null.
  • object
    1. objeto
    2. array
    3. date
    Esses três referidos serão melhores explicados em seus próprios arquivos.

Nota: Muitos daqui serão melhor abordados ou aprofundados em seus próprios arquivos.

typeof

É usado para saber qual tipo de variável ou expressão está sendo usada/trabalhada. Por exenplo:
Com typeof ("nome") o retorno será string.
Com typeof (15) o retorno será number.
