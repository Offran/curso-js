Sintaxe JS

A sintaxe JS define dois tipos de valores:
Fixos:
Também chamados de literais, podem ser números ou strings. Eles retornarão fixamente o previsto.
Os números com ou sem decimais (usando ponto e não vírgula).
E as strings (textos escritos entre áspas duplas ou simples), por exemplo:
document.getElementById("demo").innerHTML = 'John Doe'; Variáveis:
Usados para armazenar valores de dados. var, let ou const para declarar variáveis.

Operadores JS

Os operadores ( + - * /) são usados para calcular valores. Por exemplo:
(5*2)+10

Operador de atribuição (=) é usado tão somente para atribuir um valor a uma variável, como em (let x = 5;).

As expressões também podem conter valores de variáveis, como: (x + 5).
A expressão (+) serve como concatenação para strings. Por exemplo:
document.getElementById("demo").innerHTML = "John" + " " + "Doe";

Identificadores/nomes JS

Independente de como será chamada uma variável, ela deve começar sempre com uma letra (A-Z ou a-z), cifrão ($) ou sublinhado(_). Não colocar acentos ou cedilha na variável. Números não tem problema.

O javascript sempre diferencia letras maiúsculas de minúsculas, sendo Case-Sensitive, então se for escrita uma variável com maiúscula, quando ela for chamada, deverá ser exatamente igual.
Isso significa que também não poderei chamar uma variável com maiúscula caso a mesma não tenha.

Formas populares padronizadas de criar nomes de variáveis:

NomeSobrenome;
master_card;
javaScript;

Nota: javaScript usa o conjunto de caracteres chamado Unicode. Ele cobre quase todos os conjuntos de caracteres do mundo, contudo, o ideal é usar sempre o padrão previsto aqui, para evitar potenciais conflitos ou maus hábitos de programação.
