Números

Os números podem ser escrito com ou sem decimais em JS.

let x = 3.14;
let y = 4;

Número extra grandes ou pequenos podem ser escritos com notação científica:
let x = 123e5; (12300000)
let y = 123e-5 (0.000123)

Os números em JS tem precisão de até 15 dígitos:
let x = 99999999999999 (preciso)
let y = 999999999999999 (aqui fica impreciso e erra o valor)

A aritimética do float (ponto flutuante) nem sempre é 100% precisa.
let x = 0.2 + 0.1;
0.2 + 0.1 = 0.30000000000000004

Para resolver esse problema deve-se multiplicar e dividir:
let x = (0.2 * 10 + 0.1 * 10) / 10;

Números em string

Ao adicionar números em string é preciso colocar entre áspas.
let x = "20"
let y = "10"

E quando se somam strings ao invés de soma, é uma concatenação.

let z = x + y;
O resultado de z = xy

Isso acontecerá mesmo se um deles não for uma string, ele será interpretado como uma ao ser somado com uma string.

Isso é só para soma, todas as outras opções (multiplicação, divisão e subtração) funcionam normalmente, pois como apenas a concatenação tem o mesmo sinal da adição, as outras são automaticamente convertidas para number ao invés de gerar esse problema.

NaN = Not a Number

NaN é uma palavra reservada em JS que indica que um número não é um número legal/legítimo. Tentar fazer aritimética com uma string não numérica resultará em NaN (não é um número).

let x = 100 / "apple";
resultado: x = NaN

Pode ser usado a função JS isNaN() para descobrir se um valor é um número:

let x = 100 / "apple";
isNaN(x);

Vai retornar "true", pois não é categorizado como número.

Nota: NaN em um cálculo forçará o resultado a ser NaN.
typeof NaN; retorna number.

infinity

É o resultado de um cálculo acima do maior número possível.
typeof infinity retorna um number.
divisão por 0 gera infinity;

Métodos dos números:

  • toString()
    • retorna o número como string.

  • toExponential()
    • Retorna o número como notação exponencial.

  • toFixed()
    • Retorna o número com um número de decimais.
  • toPrecision()
    • Retorna um número escrito com um comprimento especificado.
  • Number()
    • Converte para number.

Existem outros métodos, maiores detalhes sobre os métodos estão aqui.

Propriedades dos números

As propriedades são usadas para representar algumas questões de numbers, por exemplo:

number.MAX_SAFE_INTERGER mostrará o maior número inteiro seguro de JS.
