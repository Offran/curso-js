O loop while

O while loop percorre um bloco de código enquanto uma condição especificada for verdadeira.
Sintaxe:

while (condition) {
// código
}

Exemplo:

let i = 1;

while (i < 10) {
text += "<br>The number is " + i;
i++;
}

Esse código acima fará um contador ir de 1 até 9.

While pode ser usado para percorrer um array

const cars = ["BMW", "Volvo", "Saab", "Ford"];

let i = 0;
let text = "";
while (cars[i]) {
text += cars[i] + "<br>";
i++;
}

document.getElementById("demo").innerHTML = text;

O loop do while

O do while é uma variante do while. Este loop executará o bloco de código uma vez antes de verificar se a condição é verdadeira, sendo essa sua diferença.

Usando break em loops

A instrução break pode ser usada para sair de um loop:

let text = "";
for (let i = 0; i < 10; i++) {
if (i === 3) { break; }
text += "The number is " + i + "<br>";
}

document.getElementById("demo").innerHTML = text;

O resultado disso é que o código irá ser um contador até o número 2.

Usando continue declaração em loops

A instrução continue pode ser usada para interromper um loop e voltar após o ponto onde foi interrompida:

let text = "";
for (let i = 0; i < 10; i++) {
if (i === 3) { continue; }
text += "The number is " + i + "<br>";
}

document.getElementById("demo").innerHTML = text;

O resultado disso é que o código irá ser um contador até o número 2, pular o 3 e então continuar no 4+.

Usando break em um array, exemplo prático

const cars = ["BMW", "Volvo", "Saab", "Ford"];

list: {
