Strings

Uma string JS contém zero ou mais caracteres escritos entre áspas. Por exemplo:

let string = "Franco";

Nota: Podem ser usadas áspas simples ou duplas.

Comprimento da string

let text = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
let length = text.length;

Essa propriedade calcula quantos caracteres estão na string, espaços também são contados, então se eu separar um espaço entre o A e o B o resultado será 27.
O resultado disso será 26.

Escrevendo nas strings

Podem ser usadas áspas dentro de uma string, desde que elas não correspondam às aspas ao redor da string (let citacao = "Let's go.";) ou (let citacao2 = 'Vou usar "aquilo"')
Observe que sempre é modificado o tipo de áspas a ser usado.

Caso o usuário queira manter as mesmas áspas então deve usar uma barra invertida "\".
Exemplos:
\' resulta em '.
\" resulta em ".
\\ resulta em \.

Isso significa que para ser corretamente usado deve ficar dessa forma:
let citacao = "Que \"merda\" você tá falando?!"

No caso da barra invertida, cada duas barras invertidas resultarão em uma, por exemplo:
\\\\ -> resulta em \\ na string.

Nota: Caso queira fazer uma única variável com uma string concatenada a ela mesma, isso também é possível. Exemplo:
(let nome = "Franco Schneider";) é o mesmo que (let nome = "Franco" + "Schneider";)

Literais de modelo JS

Literais modelos usam crase para definir uma string:

let exemplo = `Hello World!`;

Nota: Usando a crase é permitido multilinhas para uma string.

As cráses permitem citações de outras variáveis de forma mais dinâmica:
let firstName = "John";
let lastName = "Doe";
let idade = 12;


let text = `Welcome ${firstName} ${lastName}, de ${idade} anos!`;

Usando o modelo literal é possível até mesmo ajustar o JS dentro dele.

let price = 10;
let VAT = 0.25;
let total = `Total: ${(price * (1 + VAT)).toFixed(2)}`;

A substituição automática de variáveis por valores reais é chamada de interpolação de strings.

Colocando variáveis em strings

É possível colocar variáveis em strings da seguinte forma:

let header = "Templates Literals";
let tags = ["template literals", "javascript", "es6"];

let html = ` <h2>${header}</h2> <ul>`;<brgt; for (const x of tags) {
html += `<li>${x}</li>`;
}

html += `</ul>`;

Strings como objetos

Strings podem ser criadas como objetos usando:
let x = new String ("Franco";), porém não é recomendado criar objetos string, pois complica o código e diminui a velocidade de execução.
