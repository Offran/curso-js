Palavras-chave

Esse arquivo foi criado unicamente para falar de forma mais resumida sobre algumas das principais variáveis e não precisar me prender a elas nos próximos cartões.

  • var
    • Declara uma variável.

  • let
    • Declara uma variável de bloco.

  • const
    • Declara uma constante de bloco.

  • if
    • Marca um bloco de instruções a serem executadas sobre uma condição, por exemplo: "se" essa condição for atendida, execute tal comando, "se não", execute outro.

  • switch
    • Marca um bloco de instruções a serem executadas em diferentes casos, por exemplo: caso 1 = x; caso 2= y; ...

  • for
    • Marca um bloco de instruções a serem executados em um loop que repetirá o comando enquanto não for satisfeito.

  • function
    • Declara uma função (será melhor comentado quando for abordado funções.)

  • return
    • Sai de uma função.

  • try
    • Implementa tratamento de erros em um bloco de instruções (melhor abordado futuramente).
