loops for

Os loops podem exxecutar um bloco de código várias vezes.
Diferentes tipos de loops:

  • for
    • Percorre um bloco de código várias vezes
  • for/in
    • Percorre as propriedades de um objeto
  • for/of
    • Percorre os valores de um objeto iterável
  • while
    • Percorre um bloco de código enquanto uma condição específica é verdadeira
  • do/while
    • Também percorre um bloco de código enquanto uma condição especificada é verdadeira, porém ele entra no código ao menos uma vez, mesmo se a informação nunca tiver sido verdadeira

O ciclo "for"

for (let i = 0; i < 5; i++) {
text += "The number is " + i + "<br>";
}

No exemplo acima, você pode ler:
A expressão 1 define uma variável antes do início do loop (seja i = 0).
A expressão 2 define a condição para a execução do loop (i deve ser menor que 5).
A expressão 3 aumenta um valor (i++) cada vez que o bloco de código no loop é executado.

Escopo do código

let i = 5;

for (let i = 0; i < 10; i++) {
// código
}

// Resultado é 5

var i = 5;

for (let i = 0; i < 10; i++) {
// código
}

// Resultado é 10

Nota: Isso se dá pelo fato de que o var redeclara a variável.

Percorrendo um array

const cars = ["BMW", "Volvo", "Saab", "Ford", "Fiat", "Audi"];

let text = "";
for (let i = 0; i < cars.length; i++) {
text += cars[i] + "<br>";
}

document.getElementById("demo").innerHTML = text;

O iterador "i" é igual a zero, enquanto i for menor que o tamanho de "cars", percorre um valor.
Lembrando que acessando valores é a forma de percorrer um array. Esse código retornará do primeiro ao último elemento do array.

Também pode ser feito:

const cars = ["BMW", "Volvo", "Saab", "Ford"];
let i = 2;
let len = cars.length;
let text = "";
for (; i < len; i++) {
text += cars[i] + "<br>";
}

document.getElementById("demo").innerHTML = text;


Isso resultará em "Saab" e "Ford", pois começa em 2 (Saab). Além disso, note que foi excluída a parte inicial, por ter sido optado fazer antes de iniciar o for, ainda assim manteve o ponto e vírgula.

O loop for in

A instrução JavaScript for in percorre as propriedades de um objeto:

Sintaxe:

for (key in object) {
// código
}

Exemplo:

const person = {fname:"John", lname:"Doe", age:25};
let txt = "";
for (let x in person) {
txt += person[x] + " ";
}

document.getElementById("demo").innerHTML = txt;

Resultado: John Doe 25

Explicação:

  1. O loop for in itera sobre um objeto "person".
  2. Cada iteração retorna uma key (x);
  3. A key é usada para acessar o valor dela
  4. O valor da chave é person[x]

for (variável in array) {
//código
}

Não use for in em um array se a ordem do índice for importante.
Caso a ordem seja importante é melhor usar um for, for of ou Array.forEach() método.

Array forEach()

const numbers = [45, 4, 9, 16, 25];

let txt = "";
numbers.forEach(myFunction);
document.getElementById("demo").innerHTML = txt;

function myFunction(value, index, array) {
txt += value + "<br>";
}

Observe que a função leve 3 argumentos:

O valor do item
O índice do item
O próprio array

O loop for of

A instrução JavaScript for of percorre os valores de um objeto iterável.
Sintaxe:

for (variable of iterable) {
// código
}
