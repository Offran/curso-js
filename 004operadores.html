Operadores

Operadores aritiméticos

  • +
    • adição
  • -
    • subtração
  • *
    • multiplicação
  • **
    • potenciação
  • /
    • divisão
  • %
    • sobra da divisão. Por exemplo: se eu divido 5 por 2, sobra 1, então 1 será o resultado.
  • ++
    • Incrementação.
  • --
    • decrementação

Operadores de atribuição

  • =
    • (x = y)
  • +=
    • (x += y) equivale a (x = x + y)
  • -=
    • (x -= y) equivale a (x = x - y)
  • *=
    • (x *= y) equivale a (x = x * y)
  • /=
    • (x /= y) equivale a (x = x / y)
  • %=
    • (x %= y) equivale a (x = x % y)
  • **=
    • (x **= y) equivale a (x = x ** y)
  • &&=
    • Se o primeiro valor for verdadeiro, atribui um segundo. (let x=10; x&&=5;)
  • ||=
    • Se o primeiro valor for falso, atribui um segundo. (let x=undefined; x||=5;)
  • ??=
    • Se o primeiro valor for indefinido ou nulo, atribui um segundo. (let x; x??=5;)

Operadores de comparação

  • ==
    • Igual
  • ===
    • Igual em valor e tipo
  • !=
    • Diferente
  • !==
    • Diferente em valor ou tipo
  • >
    • maior quê
  • <
    • menor quê
  • >
    • maior ou igual quê
  • <
    • menor ou igual quê
  • ?

Operadores com strings

Operadores com strings se portam levando em conta que cada string conta como algo inteiro, não um número. Alguns exemplos práticos abaixo:

let z = "Hello" + 5; (string + number)
o número é contado como string e o resultado disso será Hello5. O mesmo aconteceria com outro operador matemático.

let result = "A" < "B"; (string < string)
A é menor que B? verdadeiro.
O resultado será verdadeiro se cumprir o requisito for cumprido.

Operadores lógicos

  • &&
    • "and", usado em, por exemplo: (1 && 2) para agregar os dois simultaneamentes.
  • ||
    • "or", usado em, por exemplo: (1 || 2) para agregar obrigatoriamente um ou outro.
  • !
    • "not", usado em, por exemplo: (!x) para negar que uma expressão seja verdadeira ou algum tipo de resultado.
