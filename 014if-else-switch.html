if-else e switch

if e else podem ser lidos como "se" e "se não"

  1. Use if para especificar um bloco de código a ser executado, se uma condição especificada for verdadeira
  2. Use else para especificar um bloco de código a ser executado, se a mesma condição for falsa
  3. Use else if para especificar uma nova condição a ser testada, se a primeira condição for falsa
  4. Use switch para especificar muitos blocos alternativos de código a serem executados

Sintaxes completas de if e else:

if(condição){
//bloco de código caso a condição seja verdadeira.
} else if (condição2) {
//bloco de código caso a condição 2 seja verdadeira.
} else {
//bloco de código caso nenhuma das condições anteriores seja verdadeira.
}

Nota: É preciso sempre cuidar e testar para verificar possíveis erros ou se a condição 1 pode ser pulada corretamente para a condição 2 e assim por diante. É muito comum esquecer um detalhe e algumas vezes o código parar na condição 1 em momentos imprevistos. A sintaxe pode não estar errada, mas o programador esquecer de detalhes que acabam dando o resultado incorreto para o que se espera do código.

switch

A switch é a instrução para selecionar uma das muitas opções de código a serem executadas:

switch(expression) {
case x:
// bloco de código
break;
case y:
// bloco de código
break;
default:
// bloco de código
}

É assim que funciona:

A expressão switch é avaliada uma vez.
O valor da expressão é comparado com os valores de cada caso.
Se houver uma correspondência, o bloco de código associado será executado.
Se não houver correspondência, o bloco de código padrão (default) será executado.

Exemplo:
O getDay() método retorna o dia da semana como um número de 0 a 6.
Domingo=0, Segunda=1...
O exemplo abaixo demonstra uma forma de aplicação do switch com base nisso:

let day;
switch (new Date().getDay()) {
case 0:
day = "Sunday";
break;
case 1:
day = "Monday";
break;
case 2:
day = "Tuesday";
break;
case 3:
day = "Wednesday";
break;
case 4:
day = "Thursday";
break;
case 5:
day = "Friday";
break;
case 6:
day = "Saturday";
}
document.getElementById("demo").innerHTML = "Today is " + day;

break é a palavra do código usada para sair do bloco switch. Não é necessário interromper o bloco final, pois ele termina sozinho após a leitura dele.
Ao omitir a instrução break o próximo caso será executado mesmo que a avaliação não corresponda ao caso, então ele é indispensável para evitar erros no código, apenas não sendo necessário no default.

default é a palavra chave para o padrão. Ela é o que acontecerá quando nenhuma das opções for válida.
