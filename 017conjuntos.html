Conjuntos

Um conjunto JavaScript é uma coleção de valores exclusivos.
Cada valor só pode ocorrer uma vez em um Conjunto.

Métodos de conjuntos essenciais:

  • new Set()
    • Cria um novo conjunto
  • add()
    • Adiciona um novo elemento ao conjunto
  • delete()
    • Remove um elemento de um conjunto
  • has()
    • Retorna verdadeiro se existir um valor no conjunto
  • forEach()
    • Invoca um retorno de chamada para cada elemento do conjunto
  • values()
    • Retorna um iterador com todos os valores em um conjunto
  • size
    • Retorna o número de elementos em um conjunto

Criando conjuntos e explorando métodos

// Criando um new Set
const letters = new Set(["a","b","c"]);

// Exibição do tamanho de set
document.getElementById("demo").innerHTML = letters.size;

Podemos adicionar letras com:

letters.add("d");

Isso adicionaria a letra d no array.

Podemos também adicionar variáveis, ao criá-las e depois adicionar o conjunto dessa forma:
letters.add(d);
// Caso a variável d já exista.

Nota: Caso sejam adicionadas duas variáveis de mesmo nome apenas a primeira será válida.

Iterando com value()

// Create a Set
const letters = new Set(["a","b","c"]);

// List all Elements
let text = "";
for (const x of letters.values()) {
text += x + "<br>";
}

document.getElementById("demo").innerHTML = text;

Nota: values() é o método usado para iterar todos os valores do conjunto.
