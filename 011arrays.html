Arrays/Matrizes JS

Se tiver uma lista de itens (lista de nomes de carros, por exemplo), armazenar os carros em variáveis únicas seria assim:

let car1 = "Volvo";
let car2 = "BMW"
let car3 = "Ferrari"

Agora imagine que tenhamos uma lista com 300 carros ao invés de 3?? A solução é um array.
Uma matriz pode conter muitos valores sob um único nome e você pode acessar os valores referindo-se a um número de índice.

Criando uma matriz

Usar um array literal é a maneira mais fácil de criar a matriz.

sintaxe:
const arrayName = [item1, item2, item3...]; (podem ter quebras de linhas entre os itens se o programador quiser).

Você também pode criar um array e fornecer depois os elementos:
const cars = [];
cars[0]= "Ferrari";
cars[1]= "Volvo";
cars[2]= "BMW";

Acessando elementos do array

Você acessa o elemento do array referindo-se ao número do índice:

const cars = ["Ferrari", "Volvo", "BMW"];
let car = cars[0];

Retornará Ferrari.

Lidando com arrays

Alterando um elemento de array:

const cars = ["Ferrari", "Volvo", "BMW"];
cars[0] = "Chevette";

Resultará na troca de "ferrari" por "Chevette"

Convertendo array em uma string:

cars.toString();

Caso solicitada a string resultará: Ferrari,Volvo,BMW

Acessando o array completo:

const cars = ["Ferrari", "Volvo", "BMW"];
.innerHTML = cars;

Chamando dessa forma resultará: Ferrari,Volvo,BMW

Matrizes são objetos

Matrizes são um tipo especial de objetos. O typeof operador em JS retorna "objeto" para arrays.
Objetos usam nomes pra acessar seus "membros": pessoa.primeiroNome retorna Franco
Arrays usam números para acessar seus "membros": pessoa[0] retorna Franco

Elementos da matriz podem ser objetos.

Variáveis JS podem ser objetos. Matrizes são tipos especiais de objetos. Por conta disso, você pode ter variáveis de tipos diferentes no mesmo array.
É possível ter funções, objetos e arrays em um array.

myArray[0] = Date.now;
myArray[1] = myFunction;
myArray[2] = myCars;

A propriedade do comprimento

A length propriedade de um array retorna o comprimento de um array (o número de elementos do array).

const fruits = ["banana", "laranja", "manga"];
let tamanhoDoArray = fruits.length;

Retornará 3.

Pela propriedade length podemos acessar diretamente o último número da matriz, independente de quantos ela tiver:

fruits[fruits.length - 1];

Ainda com a length propriedade:
Pode-se usar um "for" para percorrer a matriz. (for será explicado mais à frente).
Ou usar o método forEach()

Adicionando elementoa à matriz

Para adicionar elementos no array é só usar o método push().
const fruits = ["Banana", "Orange", "Apple"];
fruits.push("Lemon");

Retornará ["Banana", "Orange", "Apple", "Lemon"];

const fruits = ["Banana", "Orange", "Apple"];
fruits[6] = "Lemon"; // Isso criará um elemento undefined nos buracos não preenchidos.

Quando usar matrizes e quando usar objetos

Você deve usar objetos quando quiser que os nomes dos elementos sejam strings.
Você deve usar arrays quando quiser que os nomes dos elementos sejam números.
