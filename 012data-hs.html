Objetos de data JS

Saída de data JS

O JS tem o padrão de usar o fuso horário do navegador e exibir uma data e hora como uma sequência de texto completa.

new Date()
new Date(date string)

new Date(year,month)
new Date(year,month,day)
new Date(year,month,day,hours)
new Date(year,month,day,hours,minutes)
new Date(year,month,day,hours,minutes,seconds)
new Date(year,month,day,hours,minutes,seconds,ms)

Nota: Como os meses começam no 0, eles vão até 11, logo o mês 11 é dezembro, não novembro e o mês 1 é fevereiro, não janeiro.

const x = new Date();

Ou pode ser modificada a data:
const y = new Date("2022-03-25");
