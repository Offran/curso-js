Introdução JavaScript

No JS será explicado de forma mais pragmática qual a função de cada coisa. O JS funciona dando alvo em algo para então criar algum efeito.
Pode alterar conteúdo HTML (exemplo 1), alterar conteúdo CSS (exemplo 2). JS, diferente das outras duas linguagens, não é marcação ou estilização, mas sim de programação.

Exemplo:
O id "demo" foi adicionado nesse p, então ao clicar no botão, essa explicação será alterada.
<button type="button" onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'>Click Me!</button>

JavaScript pode alterar a estilização.

Aqui estarei alterando a fonte ao clicar no botão abaixo.
<button type="button" onclick="document.getElementById('demo2').style.fontSize='35px'">Click Me!</button>

A tag script

<script></script>
É a tag usada geralmente no final do body para fazer o script, mas pode ser colocado no head também ou até inline. Contudo, a melhor forma é fazer JS externo se não for algo muito pequeno.

O Js externo é usado <script src="myScript.js"></script>
