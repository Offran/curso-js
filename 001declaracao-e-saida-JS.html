Declarações e saídas JS

Declarações

Declarações JS criam variáveis para então podermos começar a trabalhar com elas.

Exemplos de declarações:

let x, y, z;
x = 4;
y = 6;
z = x + y;

Também é possível separar tudo em uma única linha, desde que seja respeitada a regra do ponto e vírgula, contudo o ideal é fazer de forma mais clara para um código mais limpo.

let x=4, y=6, z=x+y;

Declarando uma variável vazia:
var carName;
Com isso a declaração é automaticamente "undefined", sendo preciso colocar o siinal de igual (=) para atribuir algum valor.

Nota: Embora não seja obrigatório terminar com o ponto e vírgula, ele é altamente recomendado, assim reforçando a questão da clareza e limpeza do código. No caso de fazer em uma única linha, apenas uma vírgula é necessária na declaração.
Outra boa prática é, ao invés de fazer como na linha em que deixei tudo junto, fazer como nas outras linhas, onde cria um espaçamento para tudo ficar mais claro.

Saídas

JS pode exibir dados de diferentes maneiras:

  • innerHTML
    • Escreve em um elemento HTML
  • document.write()
    • Escrevendo na saída do elemento (após ele).
  • Window.alert()
    • Uma caixa alerta, como uma pop-up aparece no site.
  • console.log()
    • Escreve no console do navegador, não exibido no site sem inspecionar.
  • window.print()
    • Usado para abrir a opção de imprimir a página.
