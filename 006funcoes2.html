Funções 2

  • Funções são blocos de códigos reutilizáveis
    • Ou seja, podemos evitar a repetição da lógica em um programa em diversas partes do código.
  • A função precisa ser invocada para ser executada
    • Veremos na prática.

Chamando uma função sem parâmetros

function primeiraFuncao(){
console.log("Hello World das funções");
}

primeiraFuncao();

Dessa forma eu criei a função e chamei ela uma vez.

Chamando uma função com parâmetro

function dizerNome(nome){
console.log("O nome é: " + nome);
}

dizerNome("Franco");

O resultado será: "O nome é Franco"

Nota: O parâmetro aqui é como uma variável, podendo ser qualquer nome, isso facilita na compreensão do código, indicando qual deve ser o parâmetro passado.
É possível que uma variável seja um parâmetro também, por exemplo: Se eu possuo a variável x = "Felipe" e coloco como um parâmetro, essa função irá retornar "O nome é Felipe".

Chamando uma função com dois parâmetros

function soma(a, b){
let soma = a + b;
return soma;
}

let soma1 = soma(2, 3);

console.log(soma1);

O resultado será 5.
Novamente os parâmetros "a" e "b" são apenas exemplos, poderiam ser x e y.

Poderia também imprimir dessa forma:

console.log(soma(4, 5));

E imprimir o valor sem atribuir à uma variável.

Nota: A questão do return se dá pelo fato de eu atribuir parâmetros (valores)e esperar o retorno a partir deles na função.
