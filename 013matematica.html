Objeto matemático JS

Math.PI retorna a razão entre a circunferência de um círculo e seu diâmetro.
document.getElementById("demo").innerHTML = Math.PI;

Outros exemplos:

  • Math.E
    • Retorna o número de Euler
  • Math.PI
    • Retorna pi
  • Math.SQRT2
  • Math.SQRT1_2
  • Math.LN2
    • Retorna o logarítimo natural de 2
  • Math.LOG2E
    • Retorna logarítmo de Euler na base 2

Em resumo: A sintaxe para qualquer método Math é: Math.método(número)

Número para inteiro

Quatro formas de arredondar o número:

  1. Math.round(n)
    2. Arredonda para o inteiro mais próximo
  2. Math.ceil(n)
    3. Arredonda para o número inteiro maior mais próximo
  3. Math.floor(n)
    4. Arredonda para o número inteiro menor mais próximo
  4. Math.trunk(n)
    5. Retorna a parte inteira do número, sem arredondar (então sempre arredonda para baixo)

Aleatório

Math.random() retorna um número entre 0 (inconclusivo e 1 (exclusivo)), sendo sempre maior que 0 e menor que 1.

Usando métodos matemáticos ensinados acima podemos criar um número inteiro aleatório:

Math.ceil(math.random() *10);
Retorna um inteiro entre 1 e 10.

Usando uma função aleatória adequada

Como você pode ver nos exemplos acima, pode ser uma boa ideia criar uma função aleatória adequada para usar todos os fins de números inteiros aleatórios.
Essa função JS sempre retorna um número aleatório entre 1 e 10.

function getRndInteger(min, max) {
return Math.ceil(Math.random() * (max - min)) + min;
}

<button onclick="document.getElementById('demo').innerHTML = getRndInteger(0,10)">Click Me</button>

(Essa foi dificultada de graça, pois uma função sem parâmetros e multiplicada por 10 daria no mesmo).
