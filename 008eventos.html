Eventos JS

Eventos são "coisas" que acontecem aos elementos HTML. Os eventos JS "reagem" a esses eventos.
Um evento HTML pode ser algo que o navegador ou o usuário faz, por exemplo:

  • Uma página web terminou de carregar
  • Um botão foi clicado
  • Campo de entrada foi alterado

JavaScript permite executar código quando eventos são detectados.
HTML permite que atributos do manipulador de eventos, com código JS, sejam adicionados aos elementos HTML.

Exemplos:

<button onclick="document.getElementById('demo').innerHTML=Date()">The time is?</button>

Aqui é um evento onde, ao clicar, aparecerá a mensagem com a data e hora.

<button onclick="this.innerHTML=Date()">The time is?</button>

Aqui é um exemplo de evento onde "this" está sendo usado para referir a si mesmo, então um evento onde modifica o próprio elemento.

O mais comum são eventos criados a partir de funções:

<button onclick="displayDate()">The time is?</button>

function displayDate() {
document.getElementById("demo").innerHTML = Date();
}

Aqui é um exemplo onde acontece assim como no primeiro exemplo, porém está fazendo uma ligação com o JS, não diretamente inline.

Aqui uma lista de alguns eventos comuns:

  • onclick
    • Ao clicar no elemento HTML.
  • onkendown
    • Ao usuário iniciar o click no elemento HTML.
  • onchange
    • Quando algum elemento HTML for alterado (por exemplo, determinado item em uma lista foi escolhido, como uma cor de fundo e então a cor do site muda.)
  • onmouseover
    • O usuário move o mouse sobre o elemento HTML.
  • onmouseout
    • O usuário afasta o mouse do elemento HTML.
  • onload
    • Assim que o navegador terminou de carregar a página.
